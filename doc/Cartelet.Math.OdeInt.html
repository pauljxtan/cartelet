    cartelet v0.1.0 Cartelet.Math.OdeInt

    Provides functions for numerical integration of systems of ordinary differential equations.

    (N.B.: “System” in this context can refer to either systems of ODEs or the system itself that we are integrating. We’ll stick with this slightly confusing terminology until I come up with something better…)

    Link to this function integrate(ode, state_init, dt, steps, method \\ :rk4, params \\ []) 
    integrate(ode_func(), state(), float(), integer(), atom(), keyword()) ::
  {:ok, float(), state()}

    Integrates the given ODE system with the given initial conditions and step size, over a number of steps.

    If the ODE parameters and integration method are not specified, defaults to pre-defined parameters and 4th-order Runge-Kutta.

    Returns the new state.
